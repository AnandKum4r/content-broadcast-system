# Content Broadcasting System

A scalable frontend application built using React.js, Tailwind CSS, and shadcn/ui.

## Features

- Authentication
- Role-based routing
- Teacher dashboard
- Principal dashboard
- Content upload
- Approval workflow
- Public live broadcasting page
- React Query integration
- Form validation
- Responsive UI

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- shadcn/ui
- React Query
- React Hook Form
- Zod
- Axios

## Setup Instructions

```bash
npm install
npm run dev
```
